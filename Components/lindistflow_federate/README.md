# OEDISI LinDistFlow Federate

Optimal power flow using linear distflow formulation for three-phase distribution systems.

## Overview

The LinDistFlow federate implements optimal power flow (OPF) using a linearized distflow formulation. It provides efficient convex optimization for power distribution systems with support for:
- Three-phase AC power flow analysis
- Distributed generation optimization
- Voltage regulation and control
- Network constraint enforcement

## Architecture

The component is structured as:
- **EchoFederate**: Main HELICS federate class (opf_federate.py)
- **LinDistFlow Algorithm**: Linear distflow implementation (lindistflow.py)
- **Adapter**: Data conversion and network interface (adapter.py)
- **Area Management**: Zone/area partitioning support (area.py)
- **FastAPI Server**: REST API wrapper for orchestration (server.py)

## Key Features

- ✅ Linear distflow power flow algorithm
- ✅ Convex optimization using CVXPY
- ✅ Three-phase unbalanced system support
- ✅ Network graph operations with NetworkX
- ✅ Distributed generation control
- ✅ Voltage and power constraints
- ✅ HELICS co-simulation integration

```python

# Create federate with broker configuration

# Run co-simulation
federate.run()
```

### Configuration

The federate requires:
- **Static Inputs**: System parameters, network topology, control settings
- **Dynamic Inputs**: Real-time measurements via HELICS subscriptions
- **Input Mapping**: Links between HELICS subscriptions and system components

Configuration files are typically generated by the OEDISI broker based on system scenarios.

## Algorithm Details

### Linear Distflow Formulation

The linear distflow (lindistflow.py) implements:
1. **Linearized Power Flow**: Voltage drop approximation using constant voltage
2. **Network Model**: Branch admittance and incidence matrices
3. **Optimization**: Convex quadratic program solved with CVXPY
4. **Constraints**: Power balance, voltage limits, thermal limits

### Three-Phase Support

Supports unbalanced three-phase systems with:
- Phase-specific voltage and current calculations
- Neutral conductor modeling
- Asymmetric loading
